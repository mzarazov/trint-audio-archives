  • 8 May 2001

    Concordance search engine

    The organization of the Concordance interface file has been modified to use smaller individual concordance files. As stated in the notes to the interface file, the original rule was that no concordance file should exceed 50 kilobytes in size. Most of the files had grown to exceed that limit by a factor of two or more. The entries in the concordance selection table have been increased in number and have been proportioned to get concordance files that are about 40 kilobytes maximum. The reason for doing this is to avoid a longer download time each time you click on a link in the selection list.
