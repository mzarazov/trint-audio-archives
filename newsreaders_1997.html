Explanatory Remarks

How a Newsreader is Recognized

The counts are based on header lines with the following labels:

  • x-mailer
  • x-posting-agent
  • x-http-user-agent
  • x-newsreader
  • user-agent

It's possible some newsreaders use other header lines to identify themselves. The ones I've listed are the ones I've been able to learn of. If anyone knows of a different header line that's used to identify his or her newsreader, I would appreciate being told about it.

Multiple newsreaders used by a poster

If a poster used more than one newsreader during the year, the script counts only the first one. The poster's name is ignored for the rest of the year after the first time it is found.
