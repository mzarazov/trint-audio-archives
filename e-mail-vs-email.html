Preferences, "e-mail" vs "email"

by Bob Cunningham
 

Postings to AUE for the period February 1996 through 16 April 1999 have been searched to obtain a count of each of four ways of writing /'i:meIl/.

The search was performed with the Agent Newsreader. The data reside in folders in Agent, one folder for each month. The Agent Global Search function was used to perform the searches. In order to try to avoid counting postings that discussed per se the various ways of writing /'i:meIl/, postings that contained any of the four ways in the subject line were excluded from the count.

For example, for the count of postings containing 'Email', the following search string was used:

     body:Email and not subject:email and not subject:e-mail

(The logic used by Agent is such that if a search string contains any uppercase character, the search is case sensitive, while if the search string contains only lowercase characters, the search is case insensitive. Thus, the search-string element 'not subject:email' excludes both 'email' and 'Email'.)
